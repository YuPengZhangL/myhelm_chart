phone string 手机号

backphone string 备用手机号，可以为空
depart int 部门id
duty bool 是否值班，默认false

# 页面(路由)增加

# 方法：post
# 请求参数
字段名称 字段类型 是否必要 含义
parent_id int 是 父页签pid，前端指定
menu_name string 是 菜单名称
menu_symbol string 是 菜单标识 (前端使用)
menu_icon string 是 图标 (前端使用)
urls list 是 功能URL列表 数组 可以为空
action_name string 否 功能url名称
action_url string 否 功能URL地址(后端具体的路由地址)，不能为/
action_symbol string 否 功能标识(前端使用 细分到按钮层级会用到)
action_method string 否 http方法

请求例子

{
  "parent_id": -1, 
  "menu_name": "kgtest",
  "menu_symbol": "kg",
  "menu_icon": "kgtest",
  "urls": [# 功能URL列表 数组 可以为空
         {
          "action_name": "kgtest",
          "action_url": "/account/test/",
          "action_symbol": "list", 
          "action_method": "get"
        }
  	]
}

# 功能url增加/修改/删除

# 请求方法：put
# 请求参数
字段名称 字段类型 是否必要 备注
pid int 是 编辑才传
parent_id int 是 新建编辑都传 父菜单的pid
menu_name string 是 菜单名称
menu_symbol string 是 菜单标识 (前端使用)
urls list 是 功能URL列表 数组 可以为空
action_name string 否 功能URL名称
action_url string 否 功能URL地址(后端具体的路由地址)
action_symbol string 否 功能标识(前端使用 细分到按钮层级会用到)
action_method string 否 功能URL对应http方法
pid int 否 urls中的pid，存在时标识更新，不存在时标识新增

请求例子

{
    "pid": 34,
    "parent_id": -1,
    "menu_name": "kgtest",
    "menu_symbol": "kg",
    "menu_icon": "kgtest",
    "urls": [
        {
        	"pid": 46, #修改
        	"parend_id": 34,
            "action_symbol": "list",
            "action_name": "kgtest",
            "action_url": "/account/test/",
            "action_method": "get"
        },
        {#新增
            "action_symbol": "update",
            "action_name": "kgtest",
            "action_url": "/account/test/",
            "action_method": "put"
        }
    ]
}
