# 获取负载均衡器内容

查询方法：get

字段名称 字段类型 是否必要 含义
flag int 否 云平台，请查看公共数据，可选云平台，默认是腾讯云平台

响应内容

# 腾讯云
字段名称 字段类型 含义
id int 自增id
clbid string 实例ID
name string 名称
status string 状态
vip string vip
region string 所属节点
nettype string 网络类型
vpcid string 所属网络
target string 健康状态
tag list 标签列表[string,string]，可以为空
creattime datetime 创建时间，格式yyyy-mm-dd HH-MM-SS
comment string 备注
