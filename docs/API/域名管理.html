# 域名管理

# 域名信息

# 路由

/domain/info/

# 获取域名信息列表内容

查询方法：get

响应内容

字段名称 字段类型 含义
id int 自增字段
domain string 域名
provider int 运营商，0：阿里云万维网，1：aws-route53，2：非高图域名
tag list 标签列表[kg, test]
record int 记录数，前端不可修改
version string 版本
buytime date 购买时间，格式yyyy-mm-dd
expired date 到期时间，格式yyyy-mm-dd
updatetime date 域名信息更新时间，格式yyyy-mm-dd HH:MM:SS，如"2021-07-01 02:44:09"，前端不可修改
status int 域名信息状态，0(正常), 1(异常)
comment string 备注
{
    "data": [
        {
            "id": 1,
            "tag": [
                "kg",
                "hello"
            ],
            "domain": "goatgame.com",
            "provider": "阿里云",
            "record": 0,
            "version": "免费",
            "buytime": "2021-07-14",
            "expired": "2021-07-14",
            "updatetime": "2021-07-14T06:05:57.666625Z",
            "status": 0,
            "comment": null
        }
    ],
    "code": 200,
    "message": "OK"
}

    "version": "免费",
    "buytime": "2021-06-29",

}
