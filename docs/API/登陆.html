# 路由

/login/

请求方法：post

请求参数

字段名称 字段类型 是否必要 含义
username string 是 用户名
password string 是 密码

响应内容

#用户名和密码正确
{
    "code": 200,
    "message": "OK",
    "token": "eyJ0eXAiOiJa2c0NEBxcS5jb20ifQ.zIPmIcbkZB50_xAVfOxk0_V_6gyvk-UF58"
}

#用户名或密码错误
{
    "code": 598,
    "message": "{'non_field_errors': [ErrorDetail(string='无法使用提供的认证信息登录。', code='invalid')]}"
}
