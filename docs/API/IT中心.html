# 创建固产

请求方法：post

请求参数

字段名称 字段类型 是否必要 含义
ittype int 是 分类管理id
code string 是 资产编码
center string 是 所属中心
depart string 是 所属部门
cname string 否 资产名称
brand string 是 品牌
comment string 否 规格说明
cost int 是 采购价格
issuer string 否 供应商
buytime date 是 入仓时间，格式yyyy-mm-dd，默认当前日期
usetime date 是 分配时间，格式yyyy-mm-dd，默认给2099-01-01
location string 否 资产位置
status int 是 状态，默认为0(空闲)，1(已分配)，2(已废弃)
user string 否 使用人，如果status为已分配，为必填
area int 否 地区数值，默认为0，表示广州

# 导出固产内容

# 路由

/it/asset-export/

查询方法：post

请求参数

字段名称 字段类型 是否必要 备注
data list 否 待导出固产id列表[id1, id2]
area int 否 导出地区，具体值请参考固产get方法
存在，表示导出指定部分内容，否则导出所有
#导出id1和id2
{
  "data": [id1, id2],
  "area": 0
}
没有参数，表示导出所有
