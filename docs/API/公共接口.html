delete 删除

# 前端请求

类型 格式
非文件 raw json

# 后端响应

{ code: int, message: string, data: list, #[{}, {}, ] }

字段 类型 是否必要 含义
code int 是 http状态码+自定义状态码
message string 是 提示消息
data list 是 返回json列表内容，[{}, {}]
# code：状态码
code值 含义
200 成功
405 请求方法错误
598 用户名或密码错误
599 服务端数据出错
