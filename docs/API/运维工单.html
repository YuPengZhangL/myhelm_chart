# 故障统计

# 路由

/fault/groupby/

# 获取故障统计内容

查询方法：get

注意：统计中，不传递部门时，默认统计所有部门，没有传递时间时，默认统计当月

如/fault/groupby/?gbkey=flevel&start=2021-05-20&end=2021-06-01

字段名称 字段类型 含义
start date 开始时间，格式yy-mm-dd
end date 结束时间，格式yy-mm-dd
gbkey string 归类字段名称，请查看故障get请求
ftype string 故障类型
depart string 故障归属
flevel string 故障等级
issuer string 所属IDC
studio string 所属项目组
user string 处理人

响应内容

字段名称 字段类型 含义
cnt int 次数
seconds string 秒数字符串
ftype__name string 故障类型
depart__name string 故障归属
flevel string 故障等级
issuer__name string 所属IDC
studio string 所属项目组
user__realName string 处理人
{
    "data": [
        {
            "user__realName": "李海彬",
            "cnt": 1,
            "seconds": "342000.0"
        },
        {
            "user__realName": "梁健辉",
            "cnt": 1,
            "seconds": "60.0"
        }
    ],
    "code": 200,
    "message": "OK"
}
