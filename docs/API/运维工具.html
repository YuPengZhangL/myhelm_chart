        }
    }
}

# 开始初始化

请求方法：post

请求参数

字段名称 字段类型 含义 是否必要
issuer string 云商 是
mtype string 类型 是
flag bool 登陆信息标志，true：密码登陆，false: 密钥登陆 是
login dict 登陆信息字典 是
user string 用户 是
port int 端口,默认61618 否
pwd string 密码，flag为true必要 否
rsafile file 密钥文件，flag为false必要，文件内容request.FILES 否
machine dict 机器字典 是
hostname string 主机名 是
zabbixIP string zabbixIP 是
pubIP string 公网IP 是
priIP string 内网IP 是
# 请求内容
{
    "issuer": "txy",
    "mtype": "control",
    "flag": true,
    "login": {
        "user": "root",
        "port": 22,
        "pwd": "pwd12",
        "rsafile": ""
    },
    "machine": [
        {"hostname": "kg-tmp-test_01", "zabbixIP": "127.0.0.2", "pubIP": "", "priIP": "127.0.0.4"},
        {"hostname": "kg-tmp-test_02", "zabbixIP": "127.0.0.1", "pubIP": "127.0.0.1", "priIP": "127.0.0.5"}]

}
