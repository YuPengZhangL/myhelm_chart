字段名称 字段类型 是否必要 含义
flag int 是 云平台，请查看公共数据，可选云平台
name string 是 数据库名
region string 是 可用区
cpu int 是 cpu核数
memory int 是 内存大小(G)
disk int 是 磁盘大小(G)

ip string 是 内网地址

comment string 否 备注
