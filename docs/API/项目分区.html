# 项目

# 路由

/gameproject/project/

# 获取项目内容

查询方法：get

响应内容

字段名称 字段类型 含义
id int 自增字段
name string 游戏项目简写
chname string 项目中文名称
engname string 项目英文全写
tag list 标签列表[string,string]
cpname string 原厂公司名称
uptime date 上线时间，格式yyyy-mm-dd
downtime date 下线时间，格式yyyy-mm-dd
status int 项目状态，默认值为0
comment string 备注
{
    "data": [
        {
            "id": 1,
            "tag": [
                "kg"
            ],
            "name": "kg",
            "chname": "kg",
            "engname": "kg",
            "cpname": "kg",
            "uptime": "2021-06-29",
            "downtime": "2021-06-29",
            "status": 0,
            "comment": ""
        }
    ],
    "code": 200,
    "message": "OK"
}

# 创建用户管理

请求方法：post

字段名称 字段类型 是否必要 含义
utype int 是 用户类型，0：系统用户，1：管理用户，默认值为0
name string 是 管理用户名称
flag int 是 自动生成秘钥，0：自动生成，1：手动上传，默认值为0
puburl string 否 公钥上传/下载地址，flag为1时，必要1
keyurl string 否 秘钥上传/下载地址，flag为1时，必要
comment string 否 备注

响应内容

#异常响应
{
    "data": [],
    "code": 599,
    "message": {
        "name": [ErrorDetail(string='项目用户管理 with this 管理用户名称 already exists.', code='unique')]
    }
}
